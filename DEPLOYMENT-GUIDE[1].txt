# CancerianMind Website — Deployment Guide
# ==========================================
# 6 ways to deploy. Pick ONE. All are free to start.
# ==========================================

## ╔══════════════════════════════════════╗
## ║  OPTION 1 — VERCEL  (RECOMMENDED)   ║
## ║  Live in 2 minutes. Best option.    ║
## ╚══════════════════════════════════════╝

### Method A — Drag & Drop (Easiest, no code needed)
1. Go to: https://vercel.com
2. Sign up (free) with GitHub, Google, or email
3. Click "Add New Project"
4. Drag the entire cancerianmind-web/ folder into the upload area
5. Click Deploy
6. YOUR SITE IS LIVE! ✓
   URL: https://cancerianmind.vercel.app

### Method B — Via CLI
npm install -g vercel
cd cancerianmind-web
vercel
# Follow prompts → deployed in 60 seconds

### Method C — Via GitHub (best for updates)
1. Create GitHub account at github.com
2. Create new repository: cancerianmind
3. Upload all files in this folder to the repo
4. Go to vercel.com → Import Git Repository
5. Select your GitHub repo → Deploy
6. Every time you push to GitHub → auto-deploys ✓

## ╔══════════════════════════════════════╗
## ║  OPTION 6 — RENDER (with backend)   ║
## ╚══════════════════════════════════════╝

For hosting both the website AND the Python backend together:
1. Go to: https://render.com
2. Create account (free)
3. New → Static Site → upload cancerianmind-web/
4. New → Web Service → upload backend/
   Build: pip install -r requirements.txt
   Start: uvicorn main:app --host 0.0.0.0 --port $PORT
5. Add env variable: ANTHROPIC_API_KEY = your-key

## ══════════════════════════════════════
## API KEY SETUP (to enable AI chat)
## ══════════════════════════════════════

The app currently calls Anthropic API directly from the browser.
This works for testing but for production you need a secure backend.

### Quick setup (5 minutes):
1. Get API key: https://console.anthropic.com
2. Option A — Keep browser-direct (for testing only):
   Open index.html → find 'https://api.anthropic.com/v1/messages'
   This already works — just deploy!

3. Option B — Secure backend proxy (for production):
   Deploy the backend/ folder to Railway:
   - Go to railway.app → New Project → Deploy from GitHub
   - Add env: ANTHROPIC_API_KEY = sk-ant-xxxxx
   - Get URL like: https://cancerianmind-api.up.railway.app
   - In index.html replace 'https://api.anthropic.com/v1/messages'
     with 'https://cancerianmind-api.up.railway.app/chat'
